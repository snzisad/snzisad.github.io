Projects

Selected research and software systems — academic and applied.

Clinical Data Engineering
CRRT Machine Log Decryption Pipeline

Automated system to decrypt and process 22,000+ proprietary LOX binary log files from CRRT machines into structured time-series CSV data. Includes a patient record matching module linking 1,800 treatment sessions to EHR records via weighted temporal overlap algorithms.

AI Security
LabOrchestrator

An end-to-end AI security framework for medical research environments. Provides orchestrated access control, threat detection, and compliance enforcement across laboratory infrastructure using large language model–driven agents.

Privacy Compliance
ComplianceGPT

A context-aware LLM agent for automated medical data privacy compliance. Dynamically interprets regulatory context (HIPAA, IRB policies) and enforces appropriate access restrictions without manual intervention, targeting complex multi-stakeholder research environments.

Access Control
LLMAC & IPBAC

Two complementary access control frameworks: LLMAC leverages large language models for globally explainable policy enforcement, while IPBAC uses interaction provenance tracing to build privacy-aware, audit-ready access decisions across distributed systems.

Public Safety

Agentic AI framework for threat modeling to enhance public safety. Extended to trustworthy peer-to-peer firmware update protocols for autonomous drones and aircraft. Awarded Best Paper at IEEE World Forum on Public Safety Technology 2025.
